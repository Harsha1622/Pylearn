if and if-else Statements in Python

if Statement

The if statement is used to execute a block of code only when the specified condition is true. If the condition is false, the code inside the if block is skipped.

Syntax of if Statement


if condition:
    statements

Example: if Statement


age = 20

if age >= 18:
    print("Eligible to vote")

Output

Eligible to vote
