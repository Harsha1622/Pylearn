if and if-else Statements in Python

if Statement

The if statement executes a block of code only when the given condition is true.

Syntax

if condition:
    statements

Example

marks = 80

if marks >= 50:
    print("Pass")

Output

Pass

if-else Statement

The if–else statement executes one block when the condition is true and another block when the condition is false.

Syntax

if condition:
    statements
else:
    statements

Example

marks = 35

if marks >= 40:
    print("Pass")
else:
    print("Fail")

Output

Fail
