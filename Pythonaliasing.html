Aliasing in Python

What is Aliasing?

Aliasing occurs when two or more variables refer to the same object in memory. Any modification through one variable affects the same object.

Aliasing with Mutable Objects

Aliasing is commonly seen with mutable objects like lists. Assignment copies the reference, not the object itself.

Example: Aliasing with List


list1 = [10, 20, 30]
list2 = list1

list2[1] = 50

print(list1)
print(list2)
[10, 50, 30]
[10, 50, 30]

Why Does Aliasing Occur?

✔ Assignment copies references
✔ Both variables point to the same memory
✔ Changes reflect everywhere

Important Points

✔ Common with mutable objects
✔ Can cause unexpected bugs
✔ Use copy() or slicing to avoid aliasing
✔ Understanding aliasing is crucial for data safety
