Dictionary Operations in Python

What are Dictionary Operations?

Dictionary operations are actions performed on dictionaries to access, add, update, delete, check, and compare key–value pairs. Unlike lists, dictionaries use keys instead of index numbers, making data access faster and more meaningful.

Accessing Dictionary Values

Values in a dictionary are accessed using their corresponding keys. If the key exists, the value is returned directly. 


student = {"name": "Harsha", "age": 20}
print(student["name"])
Harsha

Adding and Updating Elements

If a key does not exist, a new key–value pair is added. If the key already exists, its value is updated. 


student = {"name": "Harsha"}
student["city"] = "Hyderabad"   # adding
student["name"] = "Ravi"        # updating
print(student)
{'name': 'Ravi', 'city': 'Hyderabad'}

Deleting Elements

The del keyword removes a specific key–value pair from the dictionary permanently. 


student = {"name": "Harsha", "age": 20}
del student["age"]
print(student)
{'name': 'Harsha'}

Membership Operation (in / not in)

Membership operators check the presence of keys only, not values. 


data = {"a": 1, "b": 2}
print("a" in data)
print("c" not in data)
True
True

Length of Dictionary

The len() function returns the number of key–value pairs present in the dictionary. 


marks = {"Maths": 90, "Science": 85, "English": 88}
print(len(marks))
3

Comparison of Dictionaries

Two dictionaries are equal if they contain the same keys with the same values, regardless of order. 


a = {"x": 1, "y": 2}
b = {"y": 2, "x": 1}
print(a == b)
True

Important Points

✔ Dictionaries store data as key–value pairs
✔ Keys must be unique and immutable
✔ Values can be modified
✔ Membership checks keys only
✔ Faster access compared to lists
