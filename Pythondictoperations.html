Dictionary Operations in Python

What are Dictionary Operations?

Dictionary operations are actions performed on dictionaries to access, modify, delete, check, or compare key–value pairs. Dictionaries work using keys, not index positions.

Accessing Values Using Keys

Values in a dictionary are accessed using their corresponding keys. This allows fast and direct data retrieval. 


student = {"name": "Harsha", "age": 20}
print(student["name"])
Harsha

Adding and Updating Elements

If the key does not exist, a new element is added. If the key already exists, its value is updated. 


student = {"name": "Harsha"}
student["city"] = "Hyderabad"
student["name"] = "Ravi"
print(student)
{'name': 'Ravi', 'city': 'Hyderabad'}

Deleting Elements

The del keyword is used to remove a key–value pair from the dictionary. 


student = {"name": "Harsha", "age": 20}
del student["age"]
print(student)
{'name': 'Harsha'}

Membership Operation (in / not in)

Membership operators check the presence of keys only, not values. 


data = {"a": 1, "b": 2}
print("a" in data)
print("c" not in data)
True
True

Length of Dictionary

The len() function returns the number of key–value pairs in the dictionary. 


marks = {"Maths": 90, "Science": 85, "English": 88}
print(len(marks))
3

Comparison of Dictionaries

Two dictionaries are equal if they have the same keys and corresponding values, regardless of order. 


a = {"x": 1, "y": 2}
b = {"y": 2, "x": 1}
print(a == b)
True

Important Points

✔ Dictionaries store data as key–value pairs
✔ Keys must be unique and immutable
✔ Values can be changed (mutable)
✔ Membership checks keys only
✔ Faster lookup than lists
