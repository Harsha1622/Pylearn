Jumping Statements in Python

What are Jumping Statements?

Jumping statements are used to alter the normal flow of execution in a program. Python provides break, continue, pass, and assert as jumping/control statements.

break Statement

The break statement terminates the loop immediately when a condition is satisfied. 


for i in range(1, 6):
    if i == 4:
        break
    print(i)
1
2
3

continue Statement

The continue statement skips the current iteration and moves to the next iteration of the loop. 


for i in range(1, 6):
    if i == 3:
        continue
    print(i)
1
2
4
5

pass Statement

The pass statement is a null statement. It is used when a statement is syntactically required but no action is needed. 


for i in range(3):
    pass
print("Loop completed")
Loop completed

assert Statement

The assert statement is used for debugging. It checks whether a condition is true. If the condition is false, the program stops and raises an error. 


age = 16
assert age >= 18, "Age must be 18 or above"
print("Access granted")
AssertionError: Age must be 18 or above

Important Points

✔ break exits the loop completely.
✔ continue skips the current iteration.
✔ pass does nothing but avoids syntax errors.
✔ assert helps detect logical errors during testing.
