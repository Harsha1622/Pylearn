Nested if Statement in Python

What is Nested if Statement?

A nested if statement is an if statement placed inside another if statement. It is used when a condition depends on another condition.

When to Use Nested if?

Nested if is used when multiple conditions must be checked in a specific order and the inner condition should be evaluated only if the outer condition is true.

Syntax of Nested if Statement


if condition1:
    if condition2:
        statements

Example: Nested if Statement


age = 25
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed")

Output

Entry allowed

Important Points

✔ The inner if executes only when the outer if is true.
✔ Proper indentation is very important.
✔ Nested if improves decision accuracy.
✔ Too many nested if statements reduce readability.
