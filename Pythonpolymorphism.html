Polymorphism in Python

What is Polymorphism?

Polymorphism means "many forms". It allows the same method name to behave differently depending on the object that is calling it.

Polymorphism in Python

Python supports polymorphism through method overriding, operator overloading, and functions that work with different data types.

Example: Polymorphism using Method Overriding


class Animal:
    def sound(self):
        print("Animal makes a sound")

class Dog(Animal):
    def sound(self):
        print("Dog barks")

class Cat(Animal):
    def sound(self):
        print("Cat meows")

a1 = Dog()
a2 = Cat()

a1.sound()
a2.sound()
Dog barks
Cat meows

Example: Polymorphism with Function


def add(a, b, c=0):
    return a + b + c

print(add(10, 20))
print(add(10, 20, 30))
30
60

Example: Polymorphism with Built-in Functions


print(len("Python"))
print(len([1, 2, 3, 4]))
6
4

Advantages of Polymorphism

✔ Improves code flexibility
✔ Reduces code complexity
✔ Supports method overriding
✔ Makes code reusable and scalable

Important Points

✔ Same method name, different behavior
✔ Achieved using inheritance and overriding
✔ Python supports dynamic polymorphism
✔ One of the four pillars of OOP
