String Methods in Python

What are String Methods?

String methods are built-in functions used to perform operations on strings such as modifying, searching, and formatting text. Since strings are immutable, these methods return a new string.

string.capwords()

Converts the first letter of each word to uppercase.


import string
text = "welcome to python programming"
print(string.capwords(text))
Welcome To Python Programming

upper()

Converts all characters to uppercase.


text = "Python"
print(text.upper())
PYTHON

lower()

Converts all characters to lowercase.


text = "PROGRAMMING"
print(text.lower())
programming

count()

Counts occurrences of a character or substring.


text = "banana"
print(text.count("a"))
3

strip()

Removes leading and trailing spaces.


text = "   Hello World   "
print(text.strip())
Hello World

replace()

Replaces a substring with another substring.


text = "I like Java"
print(text.replace("Java", "Python"))
I like Python

find()

Returns the index of first occurrence.


text = "python"
print(text.find("t"))
2

split()

Splits a string into a list.


text = "Python is easy"
print(text.split())
['Python', 'is', 'easy']

startswith() & endswith()


text = "Python"
print(text.startswith("Py"))
print(text.endswith("on"))
True
True
