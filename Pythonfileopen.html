File Opening Modes in Python

What are File Opening Modes?

File opening modes specify how a file is opened and what operations can be performed on it such as reading, writing, or appending.

Read Mode (r)

✔ Opens file for reading only
✔ File must already exist
✔ Default mode 


with open("data.txt", "r") as file:
    data = file.read()

Write Mode (w)

✔ Opens file for writing
✔ Creates new file if it does not exist
✔ Overwrites existing content 


with open("data.txt", "w") as file:
    file.write("Hello Python")

Append Mode (a)

✔ Adds data at the end of file
✔ File is created if not present 


with open("data.txt", "a") as file:
    file.write("\nNew Line")

Read and Write Mode (r+)

✔ Reads and writes without deleting content
✔ File must exist 


with open("data.txt", "r+") as file:
    print(file.read())

Write and Read Mode (w+)

✔ Deletes old content
✔ Creates new file if not present 


with open("data.txt", "w+") as file:
    file.write("Fresh data")

Append and Read Mode (a+)

✔ File pointer starts at the end
✔ Commonly used for log files 


with open("data.txt", "a+") as file:
    file.write("Log entry")

Binary Modes

✔ rb – Read binary file
✔ wb – Write binary file
✔ ab – Append binary file 


with open("image.jpg", "rb") as file:
    data = file.read()

Important Points

✔ Choose mode carefully to avoid data loss
✔ w and w+ erase old content
✔ r mode requires file to exist
✔ Binary mode is used for non-text files
