Introduction to File Handling in Python

What is File Handling?

File handling is the process of creating, reading, writing, and updating files stored on secondary storage. It allows programs to store data permanently.

Why File Handling is Needed?

✔ To store data permanently
✔ To read and process stored data
✔ To share data between programs
✔ To manage large amounts of information

Types of Files in Python

✔ Text files (example: .txt)
✔ Binary files (example: .bin, .dat)

Basic Steps in File Handling

✔ Open a file
✔ Perform read/write operations
✔ Close the file automatically or manually

Opening a File


file = open("sample.txt", "r")

File Opening Modes

✔ r – Read mode
✔ w – Write mode
✔ a – Append mode
✔ r+ – Read and write mode

Example: Reading a File (Recommended Way)


with open("sample.txt", "r") as file:
    data = file.read()
    print(data)
(File contents will be displayed here)

Important Points

✔ Always close files after use
✔ with statement closes file automatically
✔ File mode controls allowed operations
✔ File handling enables data persistence
