Looping Through a List in Python

What is List Loop?

Looping through a list means accessing each element one by one using looping statements such as for and while.

Using for Loop


numbers = [10, 20, 30, 40]

for item in numbers:
    print(item)
10
20
30
40

Using for Loop with Index


numbers = [10, 20, 30, 40]

for i in range(len(numbers)):
    print(numbers[i])
10
20
30
40

Using while Loop


numbers = [10, 20, 30, 40]
i = 0

while i < len(numbers):
    print(numbers[i])
    i += 1
10
20
30
40
