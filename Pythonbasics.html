Python Basics

Identifiers

Identifiers are names used to identify variables, modules, classes, and objects in Python. 

total_score = 85
user_name = "admin"

Rules for Identifiers:
✔ Letters, digits, underscore allowed
✔ Must start with letter or underscore
✔ Cannot start with digit
✔ No special symbols except underscore
✔ Keywords not allowed 

2total      # ❌ Invalid
total_2     # ✅ Valid
total@value # ❌ Invalid

Keywords

Keywords are reserved words with predefined meanings. They cannot be used as identifiers. 

if user_logged_in:
    print("Access granted")

Variables

Variables store data values. Python automatically determines data types. 

user_id = 1024
username = "developer"
account_balance = 2500.75
status = "active"
print(status)   # ✅ Correct
print(Status)   # ❌ Error

Indentation

Indentation defines code blocks in Python. Default indentation is 4 spaces. 

if account_balance > 0:
    print("Transaction allowed")
    print("Processing payment")

Comments

Comments explain code and improve readability. 

# Display welcome message
print("Welcome to PyLearn")
'''
This program checks user access
and displays a message
'''
print("Access granted")
