List in Python

What is a List?

A list is an ordered collection of elements enclosed within square brackets. Lists are used to store multiple values in a single variable.

Characteristics of List

✔ Ordered collection
✔ Allows duplicate values
✔ Mutable (modifiable)
✔ Can store different data types
✔ Indexing starts from 0

Creating a List


numbers = [10, 20, 30, 40]
print(numbers)
[10, 20, 30, 40]

List with Different Data Types


data = [1, "Python", 3.14, True]
print(data)
[1, 'Python', 3.14, True]

Accessing List Elements


numbers = [10, 20, 30, 40]
print(numbers[0])
print(numbers[2])
10
30

Important Points

✔ Lists are dynamic
✔ Elements can be changed
✔ Widely used in Python
✔ Suitable for collections of data
