Data Types in Python

What is a Data Type?

A data type defines the type of value a variable stores in memory. Python automatically assigns a data type based on the value, so no manual declaration is required.

1. Numeric Data Types

Used to store numbers. Python supports integers, floating-point numbers, and complex numbers. 


a = 10      # int
b = 3.5     # float
c = 2 + 3j  # complex

2. Sequence Data Types

Used to store collections of data in an ordered manner. Lists and tuples are mutable and immutable respectively. Strings store text. 


lst = [1, 2, 3]
tup = (10, 20)
name = "Python"

3. Set Data Type

Stores unordered, unique elements. Duplicate values are removed automatically. 


s = {1, 2, 2, 3}
print(s)

4. Mapping Data Type

Dictionaries store data in key–value pairs and allow fast access using keys. 


student = {"name": "Harsha", "language": "Python"}

5. Boolean Data Type

Used in decision making. It has only two values: True and False. 


x = True

6. None Data Type

Represents absence of a value. Commonly used for initialization. 


value = None
