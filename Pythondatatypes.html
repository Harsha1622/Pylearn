Data Types in Python

What is a Data Type?

A data type specifies the type of value a variable can hold. Python automatically determines the data type based on the value assigned to a variable.

Classification of Data Types

✔ Numeric Data Types
✔ Sequence Data Types
✔ Set Data Types
✔ Mapping Data Type
✔ Boolean Data Type
✔ None Data Type

1. Numeric Data Types


a = 10
b = 3.5
c = 2 + 3j

print(type(a))
print(type(b))
print(type(c))
<class 'int'>
<class 'float'>
<class 'complex'>

2. Sequence Data Types


lst = [1, 2, 3]
tup = (10, 20, 30)
name = "Python"

3. Set Data Type


s = {1, 2, 3, 3}
print(s)

4. Mapping Data Type


student = {"name": "Harsha", "language": "Python"}
print(type(student))

5. Boolean Data Type


x = True
print(type(x))

6. None Data Type


value = None
print(type(value))
