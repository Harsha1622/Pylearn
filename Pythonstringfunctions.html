String Functions in Python

1. len()

The len() function returns the total number of characters present in a string, including spaces. 


text = "Python"
print(len(text))
6

2. ord()

The ord() function returns the Unicode value of a single character. 


print(ord('A'))
65

3. chr()

The chr() function converts a Unicode value into its corresponding character. 


print(chr(97))
a

4. str()

The str() function converts other data types into string format. 


num = 100
print(str(num))
100

5. min()

Returns the smallest character based on Unicode value. 


text = "python"
print(min(text))
h

6. max()

Returns the largest character based on Unicode value. 


text = "python"
print(max(text))
y

Important Points

✔ These are built-in Python functions
✔ Original string is not modified
✔ min() and max() depend on Unicode values
✔ Useful for validation and character processing
