if–elif–else Ladder in Python

What is if–elif–else Ladder?

The if–elif–else ladder is used to test multiple conditions. When a condition becomes true, its corresponding block of code is executed and the remaining conditions are skipped.

When to Use if–elif–else Ladder?

The if–elif–else ladder is used when there are multiple conditions and only one block of code should be executed based on the result.

Syntax of if–elif–else Ladder


if condition1:
    statements
elif condition2:
    statements
elif condition3:
    statements
else:
    statements

Example: if–elif–else Ladder


marks = 78

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 60:
    print("Grade C")
else:
    print("Fail")

Output

Grade B

Important Points

✔ Conditions are checked from top to bottom.
✔ Only the first true condition is executed.
✔ The else block is optional.
✔ Proper indentation is mandatory in Python.
