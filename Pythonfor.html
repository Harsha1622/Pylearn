for Loop in Python

What is for Loop?

The for loop is a looping statement used to iterate over a sequence such as a list, tuple, string, or range. It executes a block of code once for each element in the sequence.

When to Use for Loop?

The for loop is used when the number of iterations is known in advance or when looping through a collection of items.

Syntax of for Loop


for variable in sequence:
    statements

Example: for Loop using range()


for i in range(1, 6):
    print(i)

Output

1
2
3
4
5

Example: for Loop with a List


languages = ["Python", "Java", "C"]

for lang in languages:
    print(lang)

Output

Python
Java
C

Important Points

✔ The loop variable takes one value at a time from the sequence.
✔ The loop continues until the sequence ends.
✔ Indentation is mandatory inside the loop.
✔ for loop improves code readability.
