Creation of Dictionary in Python

What is Dictionary Creation?

Dictionary creation means defining key–value pairs using different techniques provided by Python. Dictionaries are written using curly braces { }.

Method 1: Using Curly Braces


student = {
    "name": "Harsha",
    "branch": "CSE"
}
print(student)
{'name': 'Harsha', 'branch': 'CSE'}

Method 2: Empty Dictionary


data = {}
print(data)
{}

Method 3: Using dict()


employee = dict(name="Ravi", city="Delhi")
print(employee)
{'name': 'Ravi', 'city': 'Delhi'}

Method 4: From List of Tuples


items = [("a", 1), ("b", 2)]
print(dict(items))
{'a': 1, 'b': 2}

Method 5: Using zip()


keys = ["name", "city"]
values = ["Anita", "Mumbai"]
print(dict(zip(keys, values)))
{'name': 'Anita', 'city': 'Mumbai'}

Important Points

✔ Keys must be unique
✔ Keys must be immutable
✔ Values can be any type
✔ Dictionaries are mutable
