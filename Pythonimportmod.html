Importing Modules in Python

What is Importing a Module?

Importing a module allows a Python program to use functions, variables, or classes defined in another module. Python provides multiple ways to import modules.

Ways of Importing Modules

✔ import module_name
✔ from module_name import function_name
✔ from module_name import *
✔ import module_name as alias_name

1. import module_name


import math
print(math.sqrt(16))
4.0

2. from module_name import function_name


from math import sqrt
print(sqrt(25))
5.0

3. from module_name import *


from math import *
print(pow(2, 3))
8.0

4. import module_name as alias_name


import math as m
print(m.factorial(5))
120

Important Points

✔ A module is imported only once per execution
✔ Imported modules can be reused
✔ Avoid import * in large programs
✔ Alias names improve readability
