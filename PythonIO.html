Input and Output Statements in Python

Output Statement

Output statements display information on the screen. Python uses the print() function for output.

Sample Program: Display Output

print("Welcome to Python")
print(10 + 20)
Welcome to Python
30

Input Statement

Input statements take data from the user. Python uses input() and treats input as a string.

Sample Program: User Input

name = input("Enter your name: ")
print(name)

Formatted Output

name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Name: {name}")
print(f"Age: {age}")
Name: Alex
Age: 21
