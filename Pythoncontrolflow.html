Control Flow Constructs in Python

What are Control Flow Constructs?

Control flow constructs are statements used to control the order in which instructions are executed in a program. They determine the flow of execution based on conditions, repetition, or jumps.

Types of Control Flow Constructs

Control flow constructs in Python are classified into the following types:

1. Conditional Statements

Conditional statements are used to execute code based on a condition.

Types:
✔ if statement
✔ if–else statement
✔ if–elif–else statement
✔ nested if statement

2. Looping Statements

Looping statements are used to execute a block of code repeatedly.

Types:
✔ for loop
✔ while loop
✔ nested loops

3. Jump Statements

Jump statements are used to alter the normal flow of execution.

Types:
✔ break statement
✔ continue statement
✔ pass statement
✔ assert statement
