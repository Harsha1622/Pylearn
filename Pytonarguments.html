Function Arguments in Python

What are Arguments and Parameters?

When values are passed to a function during function calling, they are called arguments. The variables used to receive this information inside the function definition are called parameters.

Types of Arguments in Python

✔ Positional Arguments
✔ Keyword Arguments
✔ Default Arguments
✔ Arbitrary Positional Arguments (*args)
✔ Arbitrary Keyword Arguments (**kwargs)

1. Positional Arguments


def add(a, b):
    print(a + b)

add(10, 20)
30

2. Keyword Arguments


def student(name, age):
    print(name, age)

student(age=20, name="Harsha")
Harsha 20

3. Default Arguments


def greet(name="User"):
    print("Hello", name)

greet()
Hello User

4. Arbitrary Positional Arguments (*args)


def total(*numbers):
    print(sum(numbers))

total(10, 20, 30)
60

5. Arbitrary Keyword Arguments (**kwargs)


def details(**info):
    print(info)

details(name="Harsha", course="Python")
{'name': 'Harsha', 'course': 'Python'}
