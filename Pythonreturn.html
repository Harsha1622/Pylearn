Return Statement in Python

What is a Return Statement?

The return statement is used to send a value from a function back to the caller. It also terminates the execution of the function.

Why Return Statement is Used?

✔ To send computed results from a function
✔ To reuse function output in other parts of the program
✔ To stop function execution and return control to the caller

Syntax


def function_name():
    statements
    return value

Example: Function with Return Statement


def add(a, b):
    return a + b

result = add(10, 20)
print(result)
30

Return Multiple Values


def calculate(a, b):
    return a + b, a - b

x, y = calculate(20, 10)
print(x)
print(y)
30
10

Return Without Value


def show():
    print("Hello")
    return

print(show())
Hello
None

Important Points

✔ Code after return is not executed
✔ A function can have multiple return statements
✔ Return improves modularity and reusability
