Python Packages

What is a Package?

A package is a collection of related Python modules organized in a directory structure. Packages help in managing large Python applications.

Why Use Packages?

✔ Organize large programs
✔ Avoid name conflicts
✔ Improve code reusability
✔ Make programs easy to maintain

Structure of a Package


mypackage/
│
├── __init__.py
├── module1.py
└── module2.py

Creating a Package

Steps:

1. Create a folder (package name)
2. Add __init__.py
3. Add module files

Example Modules


# module1.py
def add(a, b):
    return a + b
    

# module2.py
def subtract(a, b):
    return a - b

Importing from a Package


from mypackage import module1, module2

print(module1.add(10, 5))
print(module2.subtract(10, 5))

Important Points

✔ Used in large Python projects
✔ Improves modular programming
