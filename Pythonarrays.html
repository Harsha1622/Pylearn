Introduction to Arrays in Python

What is an Array?

An array is a collection of elements accessed using index values. In traditional languages, arrays store elements of the same data type in contiguous memory locations.

Arrays in Python

Python does not have built-in arrays like C or C++. Instead, lists are commonly used to represent arrays because they are flexible and easy to use.

Creating an Array Using List


numbers = [10, 20, 30, 40, 50]
print(numbers)
[10, 20, 30, 40, 50]

Accessing Array Elements


numbers = [10, 20, 30, 40]
print(numbers[0])
print(numbers[2])
10
30

Array Using array Module


import array

arr = array.array('i', [1, 2, 3, 4])
print(arr)
array('i', [1, 2, 3, 4])

Important Points

✔ Lists are commonly used as arrays in Python
✔ Indexing starts from 0
✔ Lists can store different data types
✔ array module supports only same data type elements
