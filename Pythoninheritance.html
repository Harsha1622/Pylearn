Inheritance in Python

What is Inheritance?

Inheritance is an OOP concept where one class (child class) acquires the properties and methods of another class (parent class). It helps in code reusability and better program structure.

Syntax of Inheritance


class ChildClass(ParentClass):
    statements

Example: Single Inheritance


class Parent:
    def show(self):
        print("This is Parent class")

class Child(Parent):
    def display(self):
        print("This is Child class")

obj = Child()
obj.show()
obj.display()
This is Parent class
This is Child class

Types of Inheritance in Python

✔ Single Inheritance
✔ Multiple Inheritance
✔ Multilevel Inheritance
✔ Hierarchical Inheritance
✔ Hybrid Inheritance

Example: Multilevel Inheritance


class GrandParent:
    def gp(self):
        print("GrandParent class")

class Parent(GrandParent):
    def p(self):
        print("Parent class")

class Child(Parent):
    def c(self):
        print("Child class")

obj = Child()
obj.gp()
obj.p()
obj.c()
GrandParent class
Parent class
Child class

Advantages of Inheritance

✔ Code reusability
✔ Reduces redundancy
✔ Improves maintainability
✔ Supports method overriding

Important Points

✔ Parent class is also called base class
✔ Child class is also called derived class
✔ Python supports multiple inheritance using MRO
✔ Improves program organization
