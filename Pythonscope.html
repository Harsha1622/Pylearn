Scope of Variables in Python

What is Variable Scope?

The scope of a variable refers to the part of the program where the variable can be accessed. Python follows the LEGB rule (Local, Enclosing, Global, Built-in) to decide scope.

Types of Variable Scope

✔ Local Scope
✔ Global Scope
✔ Nonlocal Scope
✔ Built-in Scope

1. Local Scope

A variable declared inside a function is called a local variable. It can be accessed only within that function. 


def show():
    x = 10
    print(x)

show()
10

2. Global Scope

A variable declared outside all functions is called a global variable. It can be accessed anywhere in the program. 


x = 20

def display():
    print(x)

display()
20

3. Nonlocal Scope

The nonlocal keyword is used in nested functions to modify a variable from the enclosing function. 


def outer():
    x = 30
    def inner():
        nonlocal x
        x = 40
    inner()
    print(x)

outer()
40

4. Built-in Scope

Built-in scope contains predefined functions like print(), len(), and type(). 


print(len("Python"))
6

Important Points

✔ Local variables exist only inside functions
✔ Global variables are accessible throughout the program
✔ Nonlocal works only in nested functions
✔ Built-in scope is always available
