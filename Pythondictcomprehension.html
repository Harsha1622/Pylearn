Dictionary Comprehension in Python

What is Dictionary Comprehension?

Dictionary comprehension is a concise way to create dictionaries using a single line of code. It dynamically generates key–value pairs and replaces multi-line loops.

General Syntax


{key_expression : value_expression for item in iterable}

Without Dictionary Comprehension


squares = {}
for i in range(1, 6):
    squares[i] = i * i

print(squares)
{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Using Dictionary Comprehension


squares = {i: i * i for i in range(1, 6)}
print(squares)
{1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

With Condition


even_squares = {i: i * i for i in range(1, 11) if i % 2 == 0}
print(even_squares)
{2: 4, 4: 16, 6: 36, 8: 64, 10: 100}

Important Points

✔ Single-line dictionary creation
✔ Cleaner and faster than loops
✔ Supports conditions
✔ Used heavily in real-world Python
