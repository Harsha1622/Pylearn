List (Array) Methods in Python

What are List (Array) Methods?

In Python, arrays are commonly implemented using lists. List methods are built-in functions used to add, remove, search, and modify elements. Most list methods modify the original list in-place.

append()

Adds one element at the end of the list.


arr = [10, 20, 30]
arr.append(40)
print(arr)
[10, 20, 30, 40]

extend()

Adds multiple elements to the list.


arr = [1, 2]
arr.extend([3, 4])
print(arr)
[1, 2, 3, 4]

insert()

Inserts an element at a specific index.


arr = [10, 30, 40]
arr.insert(1, 20)
print(arr)
[10, 20, 30, 40]

remove()

Removes the first occurrence of a value.


arr = [10, 20, 30, 20]
arr.remove(20)
print(arr)
[10, 30, 20]

pop()

Removes and returns an element at a given index.


arr = [10, 20, 30]
arr.pop(1)
print(arr)
[10, 30]

sort()

Sorts elements in ascending order.


arr = [40, 10, 30, 20]
arr.sort()
print(arr)
[10, 20, 30, 40]
